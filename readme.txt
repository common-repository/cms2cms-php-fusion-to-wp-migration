Plugin for automated data migration from PHP-Fusion to WordPress. Convert PHP-Fusion to WordPress easily and fast.

== Description ==

With the help of CMS2CMS: PHP-Fusion to WordPress Migration Plugin you have an excellent opportunity to move your current web project a step forward towards success with minimal business disruption. Besides, no special skills or technical experience are required for successful migration.

= HOW THIS PLUGIN HELPS: =
- Easy to Use. Due to its intuitive interface, CMS2CMS service suits both experienced developers and average users to make use of;
- Fast. Because the migration process is fully automated, CMS2CMS conversion is a matter of approximately 15 min (depending on the size of a website);
- Free Demo Available – can come in handy for those who wish to see the website migration process in action and to estimate the results;
- Complete Migration price depends on the content amount you want to move to the New website;
- Data Mapping - allows you to select which data to migrate to WordPress.

= WHAT WILL BE MOVED FROM PHP-FUSION TO WORDPRESS? =
* Pages along with the content included there;
* Attachments (images and videos);
* Images included in the content body.

= PRESERVED SEO JUICE AFTER THE MIGRATION =
* Migration of SEO friendly URLs;
* Migration of keywords, metadata/meta description;
* Automated 301 redirects from existing PHP-Fusion site to the new WordPress website.


= COMPATIBLE WITH: WORDPRESS 4.1.X – 5.0.X. =
- Our support team is available live 24/7.
- For more customized migration type, CMS2CMS offers data migration service packages to order.

== Installation ==
1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose the downloaded file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the “CMS2CMS: Automated PHP-Fusion To WordPress Migration Plugin”.
5. Register an account or sign in to the existing one.
6. Provide your Existing website URL address.
7. Click ‘Verify Connection’. You’ll be redirected to the CMS2CMS website to proceed with your migration.
